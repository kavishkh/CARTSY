# Cartsy — Premium Fashion Storefront

Cartsy is a high-end, minimalist e-commerce experience designed for modern wardrobes. Built with architectural precision and a focus on tactile luxury, it provides a seamless shopping journey from discovery to checkout.

![Dynamic UI](https://images.unsplash.com/photo-1441986300917-64674bd600d8?auto=format&fit=crop&q=80&w=1200)

## ✨ Features

- **Architectural Design**: A sleek, high-contrast interface inspired by modern editorial aesthetics.
- **Smooth Navigation**: Integrated with Lenis for butter-smooth scrolling and Framer Motion for sophisticated animations.
- **Interactive Product Display**: Dynamic grids, magnetic buttons, and immersive product detail pages.
- **Mobile Optimized**: Fully responsive full-screen menus and adaptive layouts.
- **Micro-interactions**: Magnetic hover effects and scroll-triggered reveals for an engaging user experience.

## 🚀 Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll**: [Lenis](https://github.com/darkroomengineering/lenis)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: [TanStack Query](https://tanstack.com/query/latest)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) + [Shadcn UI](https://ui.shadcn.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kavishkh/CARTSY.git
   cd CARTSY
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.

## 📦 Project Structure

- `src/components`: Reusable UI elements (Header, Footer, Hero, buttons).
- `src/pages`: Main application routes (Index, Collection, ProductDetails).
- `src/lib`: Data and utility functions.
- `src/assets`: Image assets and visual resources.
