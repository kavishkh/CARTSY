CART$Y is a premium, high-fashion e-commerce storefront that redefines the digital shopping experience. Built with a focus on **tactile luxury** and **architectural precision**, it combines high-end editorial aesthetics with cutting-edge web performance.

![Hero Showcase](https://images.unsplash.com/photo-1441986300917-64674bd600d8?auto=format&fit=crop&q=80&w=1200)

---

## 💎 Core Philosophy

*   **Editorial Elegance**: Every pixel is placed with intention, drawing inspiration from high-fashion magazines and minimalist architecture.
*   **Tactile Interaction**: Using magnetic elements and physics-based animations to create a digital interface that feels physical.
*   **Fluid Motion**: Leveraging Lenis for butter-smooth scrolling and Framer Motion for sophisticated, orchestrated reveals.

---

## ✨ Key Features

### 🛍️ Immersive Commerce
- **Curated Collections**: An advanced filtering system that allows users to explore products with ease.
- **Dynamic Product Cards**: Interactive hover states and magnetic interactions for an engaging discovery phase.
- **Editorial Product Pages**: Immersive full-page layouts featuring high-resolution galleries and accordion-based specifications.

### 🎭 Visual Excellence
- **Magnetic UX**: Custom-built `MagneticButton` and interactive links that respond to proximity.
- **Smooth Navigation**: A full-screen overlay menu that orchestrates stagger animations for a premium feel.
- **Typography-First Design**: Leveraging modern font pairings and whitespace to emphasize product quality.
- **Marquee & Scrollers**: High-performance scrolling text and element reveals using `ScrollReveal` components.

### 🛠️ Technical Prowess
- **Vite & React**: Ultra-fast development and build cycles.
- **Radix UI & Shadcn**: Accessible, unstyled primitives styled with custom Tailwind tokens.
- **Lenis Smooth Scroll**: Eliminating jitter for a cinematic scrolling experience.
- **TanStack Query**: Robust state management and data fetching.

## 🚀 Getting Started

### Prerequisites

- **Node.js**: Version 18 or higher.
- **Package Manager**: `npm` or `bun` recommended.

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/kavishkh/CARTSY.git
    cd CARTSY
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # or
    bun install
    ```

3.  **Launch Development Server**
    ```bash
    npm run dev
    # or
    bun dev
    ```

4.  **Explore**
    Navigate to [http://localhost:8080](http://localhost:8080) to experience the luxury.

## 📂 Project Structure

```bash
src/
├── components/       # Reusable UI architecture
│   ├── ui/           # Radix-based primitives (Shadcn)
│   ├── MagneticButton/ # Custom interactive components
│   └── Header/Footer/Hero
├── pages/            # Application routes
│   ├── Index.tsx      # Landing page / Editorial layout
│   ├── Collection.tsx # Product discovery

```
